# Pharmacy Management System (Full SDLC Project)

A comprehensive system designed to digitalize pharmacy operations, ensuring efficient prescription processing and inventory management.

## 🚀 Project Overview
This project demonstrates a full **System Analysis and Design** phase followed by a functional web implementation.

## 📊 Phase 1: System Analysis (Documentation)
The project includes a detailed analysis phase:
- **Scenario/Report:** Business logic for customers, pharmacists, and managers.
- **Diagrams:** Context Diagrams, DFDs, and ERDs.
- **Logic Tools:** Decision Tables and Structured English for prescription verification.

## 💻 Phase 2: Implementation (Tech Stack)
- **Frontend:** Responsive UI (HTML5, CSS3, JavaScript).
- **Backend:** PHP for user registration and database interaction.

## 🛠️ Key Features
- User Authentication & Registration.
- Prescription Upload & Verification.
- Automated Chatbot for medicine inquiries.
